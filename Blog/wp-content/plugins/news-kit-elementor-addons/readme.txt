== Description ==

**All News focused** elementor widgets that you are looking for in [Elementor Page Builder](https://wordpress.org/plugins/elementor/). Currently, with 57 feature rich widgets and theme builder you can develop a complete news or blog website easily.
Theme Builder - most of you are looking for is now free in this plugin. Just navigate to the Dashboard > News Kit > Theme Builder and start building part of your website from header to footer.

[Demos & Info](https://blazethemes.com/news-kit-elementor-addons/) | [All themes](https://blazethemes.com/themes/) | [Blog](https://blazethemes.com/blog/) | [Support Forum](https://forum.blazethemes.com/news-elementor/)

**GENERIC WIDGETS**

- Site Logo & Title - Displays your site title, logo and site tagline
- Site Nav Menu - Choose from dropdown list of menus you created to display it on front
- Site Nav Mega Menu - Choose from dropdown list of menus you created and added mega menu
- Phone Call - Redirects to phone dial for ease of users
- Mailbox - Redirects to mail box for ease of users
- Back To Top - Visitors can jump to the top of the website
- Live Search - Allows to view summary results of the keywords entered in the search field
- Date & Time - Displays date and realtime clock countdown
- Random News - Redirects to the random news post
- Live Now Button - Custom button for redirection
- Theme Mode - Switch between light and dark mode
- Advanced Heading - Display your titles with advanced animations and transitions
- Tags Cloud - List out tags in your way
- Canvas Menu - Displays the chosen template in canvas popover

== Installation ==
Note : As this plugin is addon, it only works with Elementor. Make sure you have Elementor installed.
The easy way to install the plugin is via WordPress.org plugin directory.
<ol>
    <li>Go to WordPress Dashboard > Plugins > Add New</li>
    <li>Search for "News Kit Elementor Addons" and install the plugin.</li>
    <li>From file explorer > Upload the `my-plugin` directory to the "/wp-content/plugins/" directory</li>
    <li>Activate Plugin from "Plugins" menu in WordPress.</li>
</ol>

== Frequently Asked Questions ==
    = How do I use the new widgets? =
    After activating the plugin, go to any page or post editor with Elementor. You will find the new widgets under the "News Elementor Widgets" category.

    = Do I need Elementor Pro for this plugin to work? =
    No, News Kit Elementor Addons works with the free version of Elementor.

    = Can I customize the design of the widgets? =
    Yes, all widgets are highly customizable using Elementor's design settings.
    
    = What themes are supported by this plugin?  =
    News Kit Elementor Addons supports all the themes that are developed with WordPress compatibility
    
    = Does plugin widgets supported in all the themes?  =
    Yes, widgets can be used in any themes

    = Does plugin theme builder supported in all the themes?  =
    News Kit Elementor Addons theme builder supports only the themes that are developed by BlazeThemes
